   - Dark Theme: تصميم داكن للراحة البصرية

   - Yellow: #F39C12 (أصفر للحالات المعلقة)
   - Purple: #9B59B6 (بنفسجي للأطباء)
   - Pink: #E91E63 (وردي للمرضى)

🎨 كيفية الاستخدام:

1. استيراد الثيمة:
   import 'package:your_app/core/config/medical_theme.dart';

2. استخدام الألوان:
   backgroundColor: MedicalTheme.primaryMedicalBlue
   foregroundColor: MedicalTheme.pure
   color: MedicalTheme.dangerRed

3. الرسائل الملونة:
   ThemeHelper.showSuccessSnackBar(context, 'تم بنجاح');
   ThemeHelper.showErrorSnackBar(context, 'حدث خطأ');
   ThemeHelper.showWarningSnackBar(context, 'تحذير');

4. دعم الثيمين:
   color = ThemeHelper.getTextColor(context);
   isDark = ThemeHelper.isDarkMode(context);

🔄 الخطوات التالية لتحديث الملفات الأخرى:

لأي ملف جديد تريد تحديثه:

1. أضف الاستيراد:
   import '../../core/config/medical_theme.dart';

2. استبدل الألوان الثابتة:
   // قبل:
   backgroundColor: Colors.white
   
   // بعد:
   backgroundColor: MedicalTheme.pure

3. استخدم Snackbar الجديد:
   // قبل:
   ScaffoldMessenger.of(context).showSnackBar(
     SnackBar(content: Text('خطأ'), backgroundColor: Colors.red)
   );
   
   // بعد:
   ThemeHelper.showErrorSnackBar(context, 'خطأ');
